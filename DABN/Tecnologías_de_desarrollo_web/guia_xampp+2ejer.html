01

Instalación de XAMPP

1
Búsqueda del instalador
Se busca "apache xampp download" en el navegador para acceder al sitio oficial.
instalacion_xampp_cap4.png · Búsqueda en Bing
2
Página de descarga oficial
Se accede a apachefriends.org y se elige la versión para Windows (64 bit).
instalacion_xampp_cap3.png · apachefriends.org/download
3
Descargando el instalador
El navegador descarga el archivo xampp-windows-x64-8.0.30-VS16-installer.exe (≈145 MB).
cap2 · Descarga en progreso
cap1 · Ejecutar instalador
4
Advertencia UAC
Windows muestra aviso de Control de Cuentas de Usuario. Se presiona Aceptar para continuar.
instalacion_xampp_cap13.png · Advertencia UAC
5
Advertencia de Antivirus
El instalador detecta un antivirus activo que podría ralentizar la instalación. Se presiona Sí para continuar.
instalacion_xampp_cap14.png · Advertencia antivirus
6
Asistente de instalación
Se inicia el Setup Wizard de XAMPP. Se presiona Next.
instalacion_xampp_cap12.png · Setup Wizard
7
Selección de componentes
Se seleccionan los componentes a instalar: Apache, MySQL, PHP, phpMyAdmin, FileZilla, Mercury, Tomcat, Perl. Se deja la selección por defecto y se presiona Next.
instalacion_xampp_cap11.png · Select Components
8
Carpeta de instalación
Se define la ruta de instalación. Por defecto es C:\xampp. Se presiona Next.
instalacion_xampp_cap10.png · Installation folder
9
Selección de idioma
Se selecciona el idioma para el Panel de Control de XAMPP (English). Se presiona Next.
instalacion_xampp_cap9.png · Language
10
Listo para instalar
El asistente confirma que está listo para comenzar la instalación. Se presiona Next.
instalacion_xampp_cap8.png · Ready to Install
11
Instalando XAMPP
El instalador descomprime y copia todos los archivos necesarios. Se espera a que la barra de progreso complete.
instalacion_xampp_cap7.png · Unpacking files
12
Instalación completada
Setup finalizado exitosamente. Se marca la casilla "Do you want to start the Control Panel now?" y se presiona Finish.
instalacion_xampp_cap6.png · Setup Completed
13
XAMPP Control Panel activo
Se inicia el Panel de Control. Se presiona Start en Apache y MySQL. Ambos módulos quedan en verde, confirmando que el servidor está operativo en los puertos 80/443 y 3306.
instalacion_xampp_cap5.png · Control Panel v3.3.0
ℹ Para verificar el funcionamiento, hacer clic en Admin de Apache abre localhost/dashboard, y Admin de MySQL abre phpMyAdmin.
02

Ejercicio 1

🖥

Codificar eje1.php

Primer programa en PHP: mostrar un mensaje en pantalla usando la instrucción echo.
Ruta del archivo: C:\xampp\htdocs\carpeta\eje1.php

eje1.php 
<html>
<body>
  <h1>Instituto Pedro P. Diaz</h1>
  <?php
    echo "Hola Php";
  ?>
  <hr>
</body>
</html>
ℹ Las etiquetas <?php y ?> delimitan el código PHP. Todo lo que esté dentro es procesado por el intérprete antes de enviarlo al navegador.

Ejercicio 2 — Tarea

📋

Imprimir datos personales y académicos

Usando echo se muestran los datos del estudiante organizados en dos secciones con etiquetas HTML.
Ruta del archivo: C:\xampp\htdocs\carpeta\eje2.php

eje2.php 
<html>
<body>

  <h1>Datos Personales</h1>
  <hr>
  <?php
    echo "Nombre: Bryan <br>";
    echo "Edad: 18 años <br>";
    echo "Ciudad: Arequipa <br>";
  ?>

  <h1>Datos Académicos</h1>
  <hr>
  <?php
    echo "Institución: IESTP Pedro P. Díaz <br>";
    echo "Carrera: Desarrollo de Sistemas de Información <br>";
    echo "Semestre: III <br>";
    echo "Unidad Didáctica: Desarrollo de Aplicaciones Backend <br>";
    echo "Docente: Ing. Filomeno García Calisaya <br>";
  ?>

</body>
</html>
